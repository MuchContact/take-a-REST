Take a REST! API

book relation

POST

Create a booking for the room identified by the request URI

Request

Headers

The request should have the Content-Type application/json

Body

Required properties
  • checkIn: array of numbers - check in date (year, month and day)
  • checkOut: array of numbers - check out date (year, month and day)
  • includeBreakfast: boolean
Example
{
  "checkIn": [ 2014, 11, 23 ],
  "checkOut": [ 2014, 12, 1 ],
  "includeBreakfast": true
}

Responses

201 Created

Up-to-date booking state

Take a REST! API

bookings relation

GET

Returns list of bookings

Request

Headers

The request should have the Content-Type application/hal+json or application/vnd.siren+json

POST

Create a booking

Request

Headers

The request should have the Content-Type application/json

Body

Required properties
  • from: array of numbers - check in date (year, month and day)
  • to: array of numbers - check out date (year, month and day)
  • includeBreakfast: boolean
Example
{
  "from": [ 2014, 11, 23 ],
  "to": [ 2014, 12, 1 ],
  "includeBreakfast": true
}

Responses

200 OK

Up-to-date booking state

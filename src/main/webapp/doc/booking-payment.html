Take a REST! API

booking-payment relation

POST

Pays for booking

Request

Headers

The request should have the Content-Type application/json

Body

Required properties
  • cardholdersName: string - first and last name of card owner
  • creditCardNumber: string - 16 digit credit card number
  • cvv: string - 3-digit CVV code
Example
{
  "cardholdersName": "Viktor Yanukovych",
  "creditCardNumber": "1234 5678 9012 3456",
  "cvv": "123"
}

Responses

201 Created

Headers
  • Location: URI of the created booking
